## About
책을 읽으며 구절이나 느낀점을 짧게 메모하면, 독서 기록을 완성해 줍니다.  
트위터에 글을 쓰듯 짧게 조금씩 생각을 메모하면, 완독 후 나의 기록을 확인할 수 있습니다.
